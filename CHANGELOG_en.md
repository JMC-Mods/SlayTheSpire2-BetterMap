# Changelog

All important changes to this project will be recorded in this file.

Versioning rule: major.minor.patch. The major version is used for substantial feature improvements, the minor version is generally updated after a new Steam Workshop release, and the patch version is updated after each code-related commit (starting from 0).

## [1.5.0] - 2026-7-9
### Added
- Added the ability to draw on the minimap.
### Fixed
- Fixed an issue where, after entering an Ancient event, opening the map could require pressing ESC once before doodles loaded on the minimap.

## [1.4.0] - 2026-6-19
### Changed
- Migrated to the official MOD release format.

## [1.3.1] - 2026-5-15
### Fixed
- Fixed an issue where Ancient rooms at the bottom edge were not fully displayed.

## [1.3.0] - 2026-5-9
### Fixed
- Fixed an issue where the minimap could disappear after using quick save/load in an Ancient room.

### Added
- Integrated JML and added several configurable options.

## [1.2.0] - 2026-3-18
### Fixed
- Changed the packaging logic to adapt to the 0.99.1 update. (The version from a few days earlier apparently was not quite fixed.)

## [1.1.0] - 2026-3-16
### Added
- Included the JSON file in packaging to adapt to the March 13 version 0.99 update.

## [1.0.0] - 2026-3-13
### Added
- Initial release, adding a minimap to the left side of the map.
